‘Billing users’ are accounts with access to self-service billing. This article explains how to add, edit, and delete billing users, as well as assign the primary billing role.

 

Billing users versus platform users

It’s important to know that ‘billing users’ are different from ‘platform users’. A billing user has access to Self-service billing, whereas a platform user has access to the Property Platform. However, a platform user and a billing user can be linked so that the same login details can be used to log in to both platforms.

Let’s look at how billing users are created. There are two ways to add a billing user:

  • Existing – create a billing user from an existing platform user. The billing user will be linked to the platform user and will share the same login details.
  • New – allows the creation of a billing user that is not linked to an existing platform user. This user will be unable to log in to the Property Platform.

 

Billing user roles

A billing user can have one of two roles:

  • Primary billing user
  • Billing user

There must be exactly one primary billing user at all times, but there can be any number of billing users.

A primary billing user cannot be deleted. However, the role of primary billing user can be assigned to a different user.

 

Assign role of primary billing user

To assign the role of primary billing user:

  1. Click on your property name > select Billing from the dropdown menu.
  1. Click on the Billing users tab, from the list of billing users, click the edit button (pencil icon) of your chosen user, then select Assign role of primary billing user from the dropdown menu.
  1. Read the confirmation message, then click Assign

 

Create a billing user

You can give a person access to self-service billing by creating a billing user.

To create a billing user:

  1. On the Billing users tab, click Add billing user
  1. The Add billing user window allows you to add existing platform users or create a billing user separate from the property platform.
    • To add Existing users – click Existing, then a list of platform users without self-service billing access will be displayed. Select one or more platform users from the list, then click Add. A notification email will be sent to each of the users which were added.
    • To add a New user – click New, enter the new billing user’s details, then click Add. A notification email will be sent to each of the new users.

Delete a billing user

You can revoke a person’s access to self-service billing by deleting their billing user.

To delete a billing user:

  1. On the Billing users tab, click the edit button (pencil icon) of the user you wish to delete, then select Remove from the dropdown menu.
  1. Read the confirmation message, then click Remove
